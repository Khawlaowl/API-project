# My API Project

## Overview
This project is a Node.js and Express.js API for managing subscriptions and payments using Stripe. It includes Firebase for storing API keys and managing user data.

## Features
- Secure API key generation
- Stripe integration for handling subscriptions and one-time payments
- Firebase Firestore for database management
- Webhooks for handling Stripe events

## Installation
```bash
git clone https://github.com/khawlaowl/API-project.git
cd API-project
npm install
npm run dev
